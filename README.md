# 📊 2025 Job Market & Skills Demand Analysis

---
## 📌 About the Project

This project is an end-to-end data analysis of a job market dataset for the year 2025. It aims to identify trends in salaries, hiring patterns, and skill demand across different industries and locations.

By combining Python for data cleaning, SQL for analysis, and Power BI for visualization, this project highlights how data can be transformed into actionable insights for understanding market trends.

## 🛠️ Tools & Technologies

- **Python (Pandas, NumPy)** – Data Cleaning & Exploratory Data Analysis (EDA)  
- **SQL (PostgreSQL)** – Data Analysis & Aggregations  
- **Power BI** – Interactive Dashboard & KPIs  
- **Excel/CSV** – Dataset Handling
## 📂 Dataset Overview

The dataset contains 10,000 job records across emerging industries such as AI, Blockchain, Green Technology, and Quantum Computing.

Key features include:
- Salary information  
- Required skills  
- Job location  
- Remote work option  
- Company size  
- Job posting date
## 🔍 Key Business Questions Answered

- Which industries offer the highest average salaries?  
- What are the most in-demand skills in the job market?  
- How do remote jobs compare to on-site roles in terms of salary?  
- What is the impact of company size on salary?  
- Which locations have the highest number of job opportunities?  
- What are the monthly hiring trends in 2025?
## 📈 Analysis Highlights

- Analyzed salary distribution across industries to identify top-paying sectors  
- Evaluated job demand across locations to determine high-opportunity areas  
- Examined the impact of company size on salary trends  
- Identified the highest salary offered across job roles  
- Analyzed monthly hiring patterns to understand job posting trends in 2025  
- Explored overall job market distribution using key attributes like location, industry, and salary
## 📊 Power BI Dashboard

The interactive dashboard includes:

- KPI cards (Total Jobs, Avg Salary, Highest Salary, Total Locations)  
- Industry-wise average salary comparison  
- Job distribution across locations  
- Monthly hiring trend analysis for 2025  
- Interactive filter panel (Industry, Location, Remote Option, Company Size)
